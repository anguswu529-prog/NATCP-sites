# NATCP 迷你遊戲大廳

歡迎來到 NATCP 迷你遊戲大廳。這裡收錄了 15 款網頁小遊戲，不需額外安裝，透過瀏覽器即可直接遊玩。以下依照遊戲序號，為你逐一介紹各款遊戲的目標與操作方式。
## 專案架構解析
本專案建構了一個輕量化的網頁遊戲集合平台。整體架構採用單一入口路由與內嵌框架結合的設計，確保主畫面佈局的穩定性，同時讓各個獨立的遊戲模組能以模組化的方式抽換與運行。
---

## 遊戲導覽清單

遊戲的排序大致反映了操作複雜度的遞增。前段偏向單純的游標互動與基礎反射神經測試，適合快速上手。

### 1. 游標粒子
* **遊玩目標**：享受純粹的視覺回饋，無分數壓力。
* **操作方式**：在畫面上移動滑鼠或滑動手指，觀察粒子殘影跟隨游標綻放。

### 2. 點點捕手
* **遊玩目標**：精準捕捉隨機移動的目標以獲取高分。
* **操作方式**：移動游標，碰觸畫面中四竄的彩色圓點。

### 3. 無盡奔跑者
* **遊玩目標**：盡可能存活更久，挑戰動態遞增的遊戲速度。
* **操作方式**：按下空白鍵或點擊畫面進行跳躍，閃避由右至左逼近的紅色障礙物。

### 4. 拍撲鳥
* **遊玩目標**：在狹窄的上下管線縫隙中求生。
* **操作方式**：持續點擊畫面或按空白鍵，對抗重力保持飛行高度，穿梭於綠色管線之間。

### 5. 打磚塊
* **遊玩目標**：清空畫面上所有的紅色磚塊。
* **操作方式**：左右移動滑鼠控制底部擋板，反彈小球擊碎上方目標，確保小球不落入畫面底端。

經歷了基礎的物理反彈與跳躍後，接下來的幾款遊戲將引入全方位的空間壓迫感與陣列邏輯考驗，需要更多的專注力。

### 6. 彈幕閃避
* **遊玩目標**：在彈幕中尋找生機，存活時間越長分數越高。
* **操作方式**：移動游標控制藍色核心，閃避從畫面四面八方湧入、具備追蹤能力的紅色目標。

### 7. 經典貪吃蛇
* **遊玩目標**：不斷進食以延長身軀，考驗空間規劃能力。
* **操作方式**：使用鍵盤方向鍵控制綠色蛇的移動路線，吃掉紅色食物。當心撞到牆壁或自己的身體。

### 8. 記憶翻牌
* **遊玩目標**：以最少的步數找出所有隱藏的配對。
* **操作方式**：點擊覆蓋的卡牌，記下圖案位置，連續翻出兩張相同的卡牌即可消除。

### 9. 下落躲避
* **遊玩目標**：在密集落下的方塊雨中夾縫求生。
* **操作方式**：左右移動滑鼠控制底部的藍色方塊，躲避從天而降的各種大小障礙物。

### 10. 核心防禦
* **遊玩目標**：守護最後防線，精準點殺威脅。
* **操作方式**：點擊從畫面邊緣逼近的紅色目標，確保它們不會觸碰並摧毀中央的藍色核心。

最後五款遊戲打破了常規的移動限制，要求玩家在極短時間內進行動態軌跡預判與時機掌握。

### 11. 向上攀跳
* **遊玩目標**：不斷往高處攀登，避免墜落深淵。
* **操作方式**：看準左右來回移動的懸浮平台，抓準時機點擊畫面起跳，穩穩落在下一層踏板上。

### 12. 軌道閃避
* **遊玩目標**：在高速切換中平衡生存與收益。
* **操作方式**：點擊畫面左/右半邊或使用鍵盤左右鍵，在三條軌道間切換。躲避紅色障礙物的同時，盡可能收集黃色金幣。

### 13. 滑鼠切割
* **遊玩目標**：享受連續破壞的快感與風險控管。
* **操作方式**：按住滑鼠左鍵拖曳，劃出軌跡切碎拋射而出的藍色目標。漏接藍色目標或誤切紅色炸彈皆會導致失敗。

### 14. 重力反轉
* **遊玩目標**：顛覆常規空間感，在上下顛倒中求生。
* **操作方式**：點擊畫面或按空白鍵瞬間反轉重力方向，在天花板與地板之間穿梭，閃避紅色高牆。

### 15. 飛彈攔截
* **遊玩目標**：指揮防禦系統，保護底部的三座城市。
* **操作方式**：觀察敵方紅色飛彈的墜落軌跡，點擊畫面發射防禦飛彈。利用防禦飛彈抵達座標時產生的擴散爆炸波，將敵方威脅摧毀於空中。

## 🕹️ 基本操作說明

* **直覺操作**：大部分遊戲皆支援 **滑鼠點擊/拖曳** 或行動裝置的 **觸控螢幕** 操作。
* **精確控制**：部分經典遊戲（如貪吃蛇、無盡奔跑者）可使用 **鍵盤方向鍵** 或 **空白鍵** 進行精確控制。
* **快速重啟**：每款遊戲下方皆有專屬的「操作教學」提示，挑戰失敗後只需點擊畫面或按空白鍵，即可無縫重開一局，不用擔心漫長的載入時間。
